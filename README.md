# THE QUIET WEIGHT
An educational narrative web game that teaches loneliness awareness, empathy, and hidden emotional struggles through storytelling, player choices, and interactive mini-games.

About The Game

The Quiet Weight explores the idea that:

“Loneliness is not always visible.”

Players experience the lives of characters from different age groups and learn to recognize hidden signs of loneliness through everyday situations, coping behaviors, and social interactions.

The game combines:

Choice-based storytelling
Interactive mini-games
Emotional learning
Reflection and awareness
